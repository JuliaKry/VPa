# CLI-менеджер паролей

Простое CLI-приложение на Python для хранения логинов и паролей в SQLite3.
Пароли шифруются с помощью Fernet из библиотеки `cryptography`, а мастер-пароль
хранится в базе только в виде SHA-256 хеша.

## Что хранит программа

Для каждой записи сохраняются:

- название/откуда, например `Google`;
- логин;
- пароль в зашифрованном виде.

Служебные файлы:

- `passwords.sqlite3` — база данных SQLite3;
- `.key` — локальный ключ Fernet для шифрования и расшифровки паролей.

Эти файлы создаются автоматически при запуске и добавлены в `.gitignore`.

Перейдите в папку проекта:

```bash
cd /Users/uliakryneckaa/Course/VPa/Проект_3
```

Установите зависимости:

```bash
python3 -m pip install -r requirements.txt
```

Проверьте, что приложение запускается:

```bash
python3 password_manager.py --help
```

## Первый запуск

При первом запуске любой команды программа:

1. создаёт базу `passwords.sqlite3`, если её ещё нет;
2. создаёт таблицы в SQLite3;
3. создаёт файл `.key`, если его ещё нет;
4. просит придумать мастер-пароль;
5. сохраняет SHA-256 хеш мастер-пароля в базе данных.

Пример:

```bash
python3 password_manager.py list
```

После этого программа попросит:

```text
Создайте мастер-пароль:
Повторите мастер-пароль:
```

При следующих запусках нужно будет вводить этот мастер-пароль для входа.

## Команды

### Добавить новую запись

```bash
python3 password_manager.py add
```

Программа по очереди спросит:

- название сервиса;
- логин;
- пароль для сохранения.

Можно передать данные сразу через аргументы:

```bash
python3 password_manager.py add --title Google --login user@gmail.com
```

Пароль при этом будет запрошен скрытым вводом.

### Получить пароль по названию

```bash
python3 password_manager.py get Google
```

Программа попросит мастер-пароль, найдёт запись `Google` и покажет:

- название;
- логин;
- расшифрованный пароль.

### Показать список записей

```bash
python3 password_manager.py list
```

Команда показывает только названия и логины. Пароли в списке не выводятся.

### Удалить запись

```bash
python3 password_manager.py delete Google
```

Команда удаляет запись с названием `Google`.

### Создать новый пароль

```bash
python3 password_manager.py new
```

Команда генерирует новый пароль, сохраняет его в базе в зашифрованном виде и
показывает сгенерированный пароль пользователю.

Можно указать длину пароля:

```bash
python3 password_manager.py new --title GitHub --login username --length 20
```

Минимальная длина пароля — 8 символов.

## Важно

Не удаляйте файл `.key`. Без него уже сохранённые пароли нельзя будет
расшифровать.

Не передавайте файл `.key` другим людям. Он нужен для расшифровки ваших паролей.
