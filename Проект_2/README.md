# Напоминалка для macOS

Приложение на Python с интерфейсом `tkinter` и хранением напоминаний в SQLite3.

## Возможности

- Добавление напоминаний с заголовком, описанием, датой и временем.
- Хранение данных в `reminders.sqlite3`.
- Автоматическое создание таблиц при запуске.
- Просмотр всех напоминаний и фильтр по статусу.
- Удаление напоминаний.
- Перевод в статусы `Готово` и `Отменено`.
- Автоматический перевод сработавших напоминаний в `Просрочено`.
- Popup-уведомления поверх всех окон, пока приложение запущено или свернуто.

## Запуск

```bash
python3 main.py
```

Формат даты и времени в форме:

```text
YYYY-MM-DD HH:MM
```

Пример:

```text
2026-07-05 14:30
```

## Структура

- `database.py` - работа с SQLite3, создание таблицы и операции с напоминаниями.
- `app.py` - GUI, список напоминаний, фильтры, действия и popup-уведомления.
- `main.py` - точка запуска приложения.
